## Decision log

- 2026-05-20: Use Apache-2.0 for repository code. Provider data remains governed by each provider's terms.
- 2026-05-20: v0.1 includes `get_person_filmography` and `check_derivative_risk` in the public MCP tool surface.
- 2026-05-20: TMDb is implemented as a real provider with user-supplied `TMDB_API_KEY`; CI and unit/contract tests use mock or mocked HTTP only.
- 2026-05-20: KMDb and KOBIS are deferred to v0.2 after the provider abstraction is stable.
- 2026-05-20: Initial v0.1 implementation was pushed directly to `main`, so a full implementation PR and `@codex review` cannot be run retroactively without rewriting history.
- 2026-05-20: Added user-facing setup docs and a config-printing script because MCP Inspector is a developer debugging UI, not an end-user product surface.
